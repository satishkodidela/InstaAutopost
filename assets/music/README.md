# Background music for Reels

Drop royalty-free `.mp3` tracks here — the Reel generator picks one at
random each time. If the folder is empty, Reels are generated silent.

The filename is used as the music credit in the post caption, so name
files to satisfy the license's attribution requirement, e.g.
`Carefree - Kevin MacLeod (incompetech.com).mp3`.

Good sources of safe tracks: incompetech.com (CC-BY), the YouTube Audio
Library, and pixabay.com/music. Instagram's own licensed music library is
not available through the API, so never use commercial songs here.
